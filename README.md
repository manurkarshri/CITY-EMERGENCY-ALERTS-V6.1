# CITY EMERGENCY ALERTS V6.1

**Pune District Emergency Intelligence**

Production deployment for user testing.

## Release

**Version:** 6.1  
**Status:** Production User Testing Release  
**Launch Area:** Pune City, PCMC, Pune District, nearby highways and ghats

## Purpose

CITY EMERGENCY ALERTS helps citizens quickly understand what is happening, where it is happening, how serious it is, whether it affects them, what action they should take, and whether a journey is suitable.

## Tabs

Situation | Alerts | Incidents | Journey | Official | Emergency

## Intelligence layers

- Decision Intelligence
- Environmental Intelligence
- River and Dam Intelligence
- Journey Suitability Index
- Source Trust Model
- Live Source Registry
- Official-first confirmation logic
- Trusted media developing-situation support

## Source policy

Tier 1 official sources are confirmation sources.  
Tier 2 trusted media supports early awareness and developing situations.  
Unverified social media is not used for emergency confirmation.

## Deployment

Designed for GitHub Pages.

After uploading to GitHub:

1. Enable GitHub Pages from `main` branch.
2. Run `V6.1 Update All Intelligence`.
3. Run `V6.1 Validate Data`.
4. Run `V6.1 Build Status`.
